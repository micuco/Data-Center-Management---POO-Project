# Data Center Management - POO Project

This project is an implementation of a system for managing data centers and monitoring alerts.

## Description

The application simulates a monitoring system for IT infrastructure. It manages servers, user groups, and system alerts, all connected through server IP addresses.

The system processes input files and generates outputs based on different commands, similar to a real monitoring service.

## Features

- Server management (add and store servers)
- User system (User, Operator, Admin)
- Resource groups for monitoring servers
- Alert system with types and severity levels
- Command-based processing from input files
- Basic exception handling for invalid data

## Functionality

- Add servers with validation
- Create and manage monitoring groups
- Add, find, and remove group members
- Generate alerts linked to servers
- Notify groups when alerts are created

## Technologies

- Java
- OOP principles
- Design patterns

## How to run

Run the main class with the required arguments for input files. The program reads commands from files and writes results to output files.
